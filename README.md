# Cadastro Usuário

---

Este projeto é uma forma de desafiar a mim mesmo, para fins de estudo e aprimoração de conhecimento.
 
## Objetivo

- Cadastrar um novo usuário
- Editar usuário existente
- Excluir usuário existente

## Tecnologias Utilizadas

- C# ASP.NET
- Entity Framework
- SQL Server

## Pré-requisitos

- Navegador Web
